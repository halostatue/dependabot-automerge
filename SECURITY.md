# halostatue/dependabot-automerge Security Policy

## LLM-Generated Security Report Policy

Absolutely no security reports will be accepted that have been generated by LLM
agents.

## Supported Versions

This action is supported on a rolling basis. Only the most recent version will
receive security updates.

## Reporting a Vulnerability

Create a [private vulnerability report][advisory] with GitHub.

[advisory]: https://github.com/halostatue/dependabot-automerge/security/advisories/new
